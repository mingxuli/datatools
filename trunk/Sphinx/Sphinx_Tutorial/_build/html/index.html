Sphinx文档生成系统简明教程¶

Sphinx 发音[sfiŋks],斯芬克斯,即古埃及的狮身人面像。

Sphinx是一个文档生成系统，可以把reSturcturedText格式的文本转换为各种格式，包括html和pdf。一个偶然的机会发现了Sphinx， 感觉用它来写文档非常方便，但是发现网上相关的文档较少，因此决定写一个教程，详细的介绍Sphinx的使用。Google Sphinx的时候 会发现一大堆关于Sphinx的东西，但很多都是关于Sphinx Search的(一个搜索引擎)。
